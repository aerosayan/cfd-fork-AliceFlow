	Программа предназначена для расчёта поля температур в <br/>трёхмерных твердотельных моделях. В ряде случаев учитывается <br/> конвективный перенос теплоносителя. Учитываются нелинейности <br/> различной природы. Поддерживается расчёт тепловой переходной <br/> характеристики. Для ускорения вычислений используется <br/> алгебраический многосеточный метод. Для ускорения нестационарных <br/> расчётов реализованы Адаптивные Локально Измельчённые Сетки (АЛИС).
